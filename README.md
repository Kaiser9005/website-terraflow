# TerraFlow — Site Vitrine SaaS

Site marketing pour TerraFlow, l'ERP agricole intelligent pour l'Afrique.

## Stack
- React 18 + Vite 5
- GSAP + ScrollTrigger (animations scroll)
- Lenis (smooth scroll)
- 60 WebP frames (hero scroll animation Apple-style)
- Custom cursor, preloader, kinetic typography

## Sections
1. **Hero** — Canvas scroll-driven animation (60 frames Kie)
2. **Marquee** — Ticker avec skew velocity-based
3. **Fonctionnalites** — 6 cartes (Parcellaire, OHADA, IA, Tracabilite, RH, Offline)
4. **Chiffres** — 8 compteurs animes (27 modules, 99.95% uptime, etc.)
5. **Modules** — 3 onglets interactifs (RH, Agriculture, Finance) — 27 modules
6. **Tarification** — 3 plans (Starter 150 EUR, Pro 500 EUR, Enterprise sur mesure)
7. **Temoignages** — Carousel auto-rotatif
8. **Demo** — Formulaire de contact avec mailto
9. **Footer** — Navigation, modules, certifications

## Developpement

```bash
npm install
npm run dev
```

```bash

```
